# pokemonTCG

PokeVault is a responsive Pokemon TCG explorer built with React, Redux Toolkit, and RTK Query.

## Features

- Search Pokemon cards by name
- Filter by type, rarity, and set
- Sort by name or HP
- Detailed card pages with attacks, weaknesses, set info, and pricing
- Favorites collection powered by Redux Toolkit
- RTK Query data fetching and caching
- Dark theme with optional light mode toggle
- Vercel-ready SPA routing

## Run locally

```bash
npm install
npm run dev
```
